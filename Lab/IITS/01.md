# 1. Study of Browser Features, Browsing, Use of Various Search Engines and Writing Search Queries
- **Software:** 
    1. Computer system 
    2. Operating system (windows/linux)
    3. Web browser (Google Chrome/ Mozilla Firefox/ Microsoft Edge)
    4. Internet Connection 

A web browser is a software application used to access and view websites on the internet. It allows users to retrive information from web servers and display it in the form of web pages. Browsers provide various features such as tabbed browsing, bookmarks, downloads, extensions and security options. 

Web browsing is the process of navigating through web pages using hyperlinks, URLs and search engines. Search engines are web based tools that help users find information on the internet by entering keywords and queries. Common search engines include Google, Bing, Yahoo and Duckduckgo. 

A search query is a set of words or phrases entered into a search engine to find specific information. Writing a proper search query helps in getting accurate and relevant results. 

## Steps 
1. Switch on the computer and connect it to the internet. 
2. Open a web browser such as Google Chrome or Mozilla Firefox. 
3. Observe the main features of the browser like address bar, tabs, bookmarks, history and settings. 
4. Enter a website address (URL) in the address bar and browse the internet. 
5. Open a search engine such as Google, Bing or Yahoo. 
6. Type different search queries related to a topic and observe the search results. 
7. Modify the search queries by adding or removing keywords to get better results. 
8. Close the browser after completing the experiment. 

## Observations 
1. Different browsers provide similar basic results with slight differences in interface. 
2. Search engine display results based on keywords entered by the user. 
3. More specific search queries give more basic results. 

## Result 
1. Thus, the features of web browsers, the process of web browsing, the use of various search engines and the method of writing effective search queries were studied successfully. 
2. Use of a secure and updated web browser is important. 
3. Avoid visiting unknown or unsafe websites. 
4. Do not share personal information on untrusted websites.

# 3. Computer Hardware Components and Identification of Ports 
## Introduction 
Computer hardware refers to the physical parts of a computer which can be seen and touched. 

## Main Hardware Components 
1. **Monitor:** displays output 
2. **CPU (Central Processing Unit):** brain of the computer 
3. **Keyboard:** used for typing input 
4. **Mouse:** used for pointing and selecting 
5. **Printer:** used to print documents 
6. **Scanner:** used to scan documents 

## Internal Components
- RAM (Random Access Memory)
- Hard disk / SSD 
- Motherboard 
- Power Supply Unit 

## Various Ports and Interfaces 
1. **USB Port:** connect pen drive, keyboard, mouse, etc. 
2. **HDMI Port:** connect monitor or projector
3. **VGA Port:** connect display devices 
4. **Ethernet Port:** connect ethernet cable 
5. **Audio Jack:** connect headphones or speakers 

## Related Cables 
- USB cable 
- HDMI cable 
- Power cable 
- LAN cable 

## Conclusion 
Understanding computer hardware and ports help in proper connection and maintenance. 

# 4. Word Processing using Openoffice Writer 
## Introduction
Word processing is the process of creating, editing, formatting, and printing text document using a computer. Openoffice writer is a word processing softwre used to prepare letters, reports, assignments and official documents. It provides various tools for text formatting and document management. 

## 1. Creating a new document 
- **Steps**
    1. Open Openoffice Writer 
    2. Click on file in the menu bar 
    3. Select new 
    4. click on text document 
    5. A new blank document will open 
    6. Type the required text 

## 2. Saving a document 
- **Steps** 
    1. Click on file 
    2. Select 'Save as'
    3. Choose the location (Desktop or folder)
    4. Enter a file name 
    5. Click save 

The document will be saved in .odt format. 

## 3. Editing text (cut, copy, paste)
- **Steps** 
    1. Select the text 
    2. Click on edit 
    3. Choose: cut, copy, paste 
    4. The selected text will be edited accordingly 

## 4. Text formatting 
1. **Changing font style and size**
    1. Select the text 
    2. Click on format 
    3. Choose character 
    4. Selected font types, size and style 
    5. Select OK 
2. **Changing alignment**
    1. Select the text 
    2. Click on format 
    3. Select paragraph 
    4. Choose alignment: left, center, right, justified 
    5. Click OK 
3. **Superscript or Subscript**
    - Superscript is used in power $(x^2)$
    - Subscript is used for chemical formulas $(H_2O)$
    - **Steps** 
        1. Select the character 
        2. Click on format 
        3. Choose character 
        4. Go to position tab 
        5. Select Superscript or Subscript
        6. Click OK 
## 5. Bullets and Numbering 
- **Steps** 
    1. Select the text 
    2. Click on format 
    3. Choose bullets and numbers 
    4. Select the required style 
    5. Click OK 

## 6. Printing a Document 
- **Steps** 
    1. Click on file 
    2. Select print 
    3. Choose printer settings 
    4. Click print 

## Result 
The word processing operations such as creating, saving, editing and formatting text were successfully performed using Openoffice Writer. 

## Conclusion 
Openoffice Writer is an important word processing tool used to create and format documents. It provides various features such as text formatting, alignment, bullets, numbering and printing which helps in preparing professional documents.
