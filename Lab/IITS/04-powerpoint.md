# Presentation using OpenOffice Impress / MS PowerPoint  

## Introduction  
Presentation software is used to create slide-based presentations for communicating ideas effectively. MS PowerPoint and OpenOffice Impress are commonly used presentation tools. They allow users to combine text, images, charts, audio and video in the form of slides. These presentations are widely used in schools, colleges, offices and seminars.  

## Objectives  
- To create and design slides  
- To use formatting tools for better presentation  
- To apply transitions and animations  
- To present information clearly using slides  

## Requirements  
1. Computer system  
2. Operating system (Windows/Linux)  
3. MS PowerPoint / OpenOffice Impress  

## 1. Starting the Presentation Software  
- **Steps**  
    1. Click on Start Menu  
    2. Select MS PowerPoint / OpenOffice Impress  
    3. Choose Blank Presentation  
    4. A new slide will open  

## 2. Creating Slides  
- **Steps**  
    1. Click on New Slide  
    2. Select the required layout  
    3. Enter title and content  
    4. Add multiple slides as required  

## 3. Formatting Slides  
- **Steps**  
    1. Select the text or object  
    2. Use formatting options (font, size, color)  
    3. Apply themes from Design tab  
    4. Adjust alignment and layout  

## 4. Inserting Objects  
- **Steps**  
    1. Click on Insert menu  
    2. Add pictures, shapes, charts or tables  
    3. Insert audio or video if required  
    4. Resize and arrange objects properly  

## 5. Applying Transitions  
- **Steps**  
    1. Click on Transitions tab  
    2. Select a transition effect  
    3. Apply to selected slide or all slides  
    4. Adjust duration and effect options  

## 6. Applying Animations  
- **Steps**  
    1. Select text or object  
    2. Click on Animations tab  
    3. Choose animation effect  
    4. Set timing and order of animation  

## 7. Slide Show  
- **Steps**  
    1. Click on Slide Show  
    2. Select From Beginning  
    3. Use keyboard or mouse to navigate  
    4. Press Esc to exit slide show  

## 8. Saving the Presentation  
- **Steps**  
    1. Click on File  
    2. Select Save As  
    3. Choose location  
    4. Enter file name  
    5. Click Save  

The presentation is saved in .pptx or .odp format.  

## Observations  
1. Slides help in presenting information clearly and visually.  
2. Transitions and animations improve presentation quality.  
3. Multimedia elements make presentations more attractive.  

## Result  
The presentation was successfully created using slides, formatting tools, transitions and animations.  

## Conclusion  
Presentation software like MS PowerPoint and OpenOffice Impress is useful for creating effective and attractive presentations. It helps in organizing ideas clearly and presenting information in a professional manner.
