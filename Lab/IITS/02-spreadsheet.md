## Introduction 
A spreadsheet is an electronic document used to organize, store and analyze data in tabular form. TI consists of rows and columns that form cells, where data can be entered and manipulated using formulas and functions. Spreadsheet software helps users perform calculations quickly, create charts and manage information efficiently. 

Examples of spreadsheet software are Microsoft Excel, Google Sheets and Libreoffice Calc. 

## Basic components of a Spreadsheet 
1. **Workbook:** a workbook is the main spreadsheet file. It contains one or more worksheets. 
2. **Worksheet:** a worksheet is a single page in a workbook where data is entered. It consists of rows and columns. 
3. **Rows:** rows are horizontal lines in a worksheet. They are identified by numbers (1, 2, 3, ....).
4. **Columns:** Columns are vertical lines in a worksheet. They are identified by letters (A, B, C, ....). 
5. **Cell:** a cell is the intersection of a row and a column. Each cell has a unique address called a cell reference (e.g., A1, B2, C3). 
6. **Cell address:** the location of the cell is called the cell address. It is formed by combining the column letter and row numbers. E.g., A1 $\rightarrow$ column A, row 1.

## Types of Data in Spreadsheet 
1. **Text (label):** words or letters (e.g., Name, Address)
2. **Numbers (value):** numeric data (e.g.: 100, 250)
3. **Formula:** used to perform calculations (e.g.: A1 + B1) 
4. **Data and Time:** Example: 12/02/2026, 10:30 AM 

## Formula and Function 
- **Formula:** A formula is an expression used to calculate values, it always starts with an equal sign (=). E.g.: `=A1+B1`
- **Function:** A function is a predefined formula that perform specific calculations. E.g.: 
    - SUM(): adds numbers 
    - AVERAGE(): finds average 
    - MAX(): finds maximum value 
    - MIN(): finds minimum value 
    - E.g., `=SUM(A1+A5)`
