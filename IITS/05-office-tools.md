# Office Tools 
- MS Word = OpenOffice Writer 
- MS Excel = OpenOffice Impress
- MS Powerpoint = OpenOffice Impress 

Office tools are a type of application software that assists users in regular office jobs like creating, updating and maintaining documents. It helps us in handling large amount of data. 

- Some of the softwares that do these are: 
    1. Word processor 
    2. Spreadsheet 
    3. Database systems 
    4. Presentation software 
    5. Email tools 

The office productivity software suite is available in many languages and is compatible with all major operating systems like Windows, Mac and Linux. It includes 4 major applications- Writer, Calc, Impress and Draw.

| MS Office | OpenOffice |

## OpenOffice Writer 
- .odt: default file extension

- Creating new document: File -> New -> Text Document 
- Open existing document: File -> Open -> Choose wanted document 
- Saving a document: File -> Save 
- Saving changes: Save button toolbar 
- Viewing in Writer: File -> Page Preview 

> Click zoom button in standard toolbara to zoom 

- Printing document: File -> Print **OR** Print button in standard toolbar
- Undo and Redo changes: Undo/Redo button in standard toolbar **OR** Ctrl+Z and Ctrl+Y respectively 

### Working with Text 
- Select all: Ctrl+A 
- Formatting characters and paragraphs: Insert -> Special character -> Choose required character 
- Insert table: table button in toolbar -> Choose required rows and columns 
- Inserting rows and columns 
    - Rows: click insert row icon in table menu bar **OR** Right click -> Row -> Insert **OR** Table -> Insert -> Row 
    - Column: click insert column icon in table menu bar **OR** Right click -> Column -> Insert **OR** Table -> Insert -> Column 
    - Deleting row: Right click -> Row -> Delete **OR** Table -> Delete -> Row 
    - Deleting column: Right click -> Column -> Delete **OR** Table -> Delete -> Column 
- Splitting cell: split cells icon in table menu bar 
- Merging cell: click merge cells icon in table menu bar 

## OpenOffice Impress 
OpenOffice Impress is openoffice.org's presentation component. With this we can create slides with many different elements including text, bulleted and numbered lists, tables, charts, clip art and a wide range of graphic objects. 

- How to start Impress 
    1. From system menu or openoffice.org quick launcher 
    2. File -> New -> Presentation 

### Parts of Impress Window 
1. **Slides Pane:** Contains thumbnail pictures. Additional operations that can be performed are: 
    1. Add new slide 
    2. Hide a slide 
    3. Delete a slide 
    4. Rename a slide 
    5. Copy slide 
    6. Change sequence of slides 
2. **Workspace:** Has 5 views:
    1. Normal: it is the main view for creating individual slides. We use this view to format and design slides. 
    2. Outline: shows topic titles, bullet lists and numbered lists for each slide in outline format. 
    3. Notes: lets you to add notes to each slide that are not seen when the presentation is shown. 
    4. Handout: lets you print your slides in a handout. 
    5. Slide sorter: shows the thumbnail of each slide in order. We use this view to rearrange the order of slides. 
3. **Tasks Pane:** It has the following sections:
    1. Master pages (slide design for presentation) 
    2. Custom animation 
    3. Slide transition 
    4. Styles and format 
    5. Gallery 
    6. Animator
