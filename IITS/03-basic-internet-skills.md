# Basic Internet Skills 
## Web Browser 
A web browser is an application software used to access information on the world wide web. It allows users to open websites, visit webpages, download files, watch videos and interact with online services.  
A browser communicates with web servers using internet protocols using HTTP and HTTPS, to retrieve information and display it in a readable format. It also manages bookmarks, browsing history, cookies, downloads and saved passwords, making internet use more convenient. E.g., firefox, edge, chromium. 

## Search Engine 
A search engine is a web based tool that helps users find information on the internet by entering keywords or phrases. A search engine scans billions of webpages and returns relevant results based on algorithms. E.g., google, brave, bing, etc.  

To use search engine efficiently- 

1. Use specific keywords instead of long sentences. 
2. Use minus sign to exclude unwanted terms. 
3. Use filters such as date, images or pdf to refine results.
